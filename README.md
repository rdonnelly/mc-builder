# mc-builder

A Marvel Champions Deck Builder

## Getting Started

- `nvm install`
- `nvm use`
- `npm -g install yarn`
- `rbenv install`
- `gem install bundler`
- `bundle install --gemfile=packages/mobile/Gemfile`

## Mobile

- Development iOS
  - `yarn dev:ios`
- Development Android
  - `yarn dev:android`
- Serve
  - `yarn serve:mobile`

## Web

- Development
  - `yarn dev:web`

## Other Commands

- Lint
  - `yarn lint`
- Check Types
  - `yarn ts`
- Clean Build Directories
  - `yarn clean`
